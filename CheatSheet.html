      Web Server
      1. What is a web server?
          Basically, a web server is an expensive rack (this is the case where they are stored above of each other) mountable PC,
          which is mostly located at a server farm where the Internet connection is extremely fast.
      2. What does a web server?
          The job of a Web Server is to serve you a webpage which is shared with his web server software.
          If it is a more complex webpage it also will have a kind of IO which means: you can log In or press buttons.
      3. How can a content be stored on a web server?
          The files that are stored on a webserver are stored in a data base or simply as a file on the desktop.
      4. How can contents in webpages described?
          In HTML or in combination with some flash player software ore scripting language for making the webpage more lifely.

      Web Client
      1. What is a web client?
          A web client is every device that is connected to the internet.
      2. What does a web client?
          A web client sends data to servers that will analyze the request and serve the client with the requested information.
      3. Web Client and Web Browser is the same?
          No, as I Sayed the client will only send and receives packages,
          and the web browser will display this content and manage what the client should request next from the server.
